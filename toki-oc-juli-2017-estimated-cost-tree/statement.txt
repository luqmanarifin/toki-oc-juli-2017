Path Ganjil

[Deskripsi]

Lelah membaca deskripsi soal TOKI Open Contest yang terlalu panjang, Anda memutuskan mencari soal dengan deskripsi pendek. Beruntung Anda menemukan soal ini!

Diberikan sebuah graph berbentuk tree dengan weighted undirected edge. Path didefinisikan sebagai jejak telusur edge dimulai dari suatu vertex dan berakhir pada vertex lain tanpa ada edge yang dikunjungi lebih dari sekali. Ongkos suatu path didefinisikan sebagai total cost edge yang dilewati pada path tersebut. 

Berapa jumlah dari seluruh path yang mungkin yang memiliki ongkos ganjil?

[Format Masukan]

Baris pertama berisi satu buah bilangan bulat N, berapa banyak vertex pada tree. Baris kedua hingga baris ke-N, masing-masing berisi 3 buah bilangan bulat U_i, V_i, C_i, artinya ada edge tanpa arah antara vertex ke U_i dan V_i dengan nilai beban C_i.


[Format Keluaran]

Keluarkan sebuah bilangan bulat, jumlah dari seluruh path yang mungkin yang memiliki ongkos ganjil.


[Contoh Masukan 1]

3
1 2 2
2 3 1


[Contoh Keluaran 1]

8


[Penjelasan]

Ada 9 kemungkinan path:
1 -> 1, biaya 0
1 -> 2, biaya 2
1 -> 3, biaya 3
2 -> 1, biaya 2
2 -> 2, biaya 0
2 -> 3, biaya 1
3 -> 1, biaya 3
3 -> 2, biaya 1
3 -> 3, biaya 0

Total path dengan ongkos ganjil adalah: 3 + 1 + 3 + 1 = 8


[Batasan]

- 1 ≤ N ≤ 10^5
- 1 ≤ U, V ≤ N
- 1 ≤ C ≤ 100
